This web app also provides a platform to seek advice and recommendations from verified nutritionalists and health coaches regarding lifestyle.

## Get it Running

```sh
yarn install
yarn dev
```

**NOTE** : Add MongoDB URI in a `.env` file at project root, like this (or pass it as an environment variable):

```env
URI=mongodb+srv://username:password@cluster.url/db?retryWrites=true&w=majority
```
